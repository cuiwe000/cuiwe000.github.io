在mac上搭建Hexo

Hexo 原理:（引自唐巧）

  • Hexo 的原理和 Octopress 一样，都是生成静态文件，这样可以方便地托管到 GitHub 和 GitCafe 上。
  • Hexo 是基于 Node 的，而 Node 的速度非常快。
